Fotoresistencia y LED

Encendido automático dependiendo de la luz

📋 Descripción

El LED se encenderá cuando haya poca luz y se apagará cuando haya mucha luz, utilizando una fotoresistencia (LDR).

🛠️ Materiales

  • Fotoresistencia (LDR)
  • LED
  • Resistencia 220Ω (para LED)
  • Resistencia 10kΩ (para LDR)
  • Protoboard
  • Cables (jumpers)
  • Arduino
1

Conexión de la LDR

Conecta la fotoresistencia a 5V y al pin A0.
Agrega una resistencia de 10kΩ a GND (divisor de voltaje).

2

Conexión del LED

Ánodo del LED → pin digital 9
Cátodo → resistencia 220Ω → GND

3

Funcionamiento

El Arduino lee la luz desde la LDR. Si hay poca luz → enciende el LED. Si hay mucha luz → apaga el LED.

💻 Código

int ldr = A0; int led = 9; int valor = 0; void setup() { pinMode(led, OUTPUT); Serial.begin(9600); } void loop() { valor = analogRead(ldr); Serial.println(valor); if(valor < 500){ // poca luz digitalWrite(led, HIGH); }else{ // mucha luz digitalWrite(led, LOW); } delay(500); }
