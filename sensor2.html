Servo Motor & Sensor Ultrasonico

Uso del Servo Motor y Sensor Ultrasonico

📋 Descripción

En la practica de hoy se realizara el movimiento de un Servo Motor dependiendo la distancia que este del Sensor Ultrasonico

🎬 Video del Proyecto

🛠️ Materiales Necesarios

🤖 Sensor ultrasonico
📋 1 Protoboard
🔴 LEDS
🛎️ Resistencias
🚠 Jumpers macho-macho
1

Inserta el sensor en la protoboard

💡
2

Coloca los tres leds en la protoboard de igual forma, pero algo alejados del sensor.

3

Coloca las resistencias. Llevalas del cátodo del led al lado negativo de la protoboard.

⚠️ Tip: El Cátodo es la patita más corta del led.
4

Conecta los jumpers del sensor ultrasonico.

📏 TTip: VCC siginifica 5V. Por lo que siempre ira conectado ahí, o a la parte positiva de la protoboard.
5

Conecta los jumpers de los leds y las resistencias a la placa Arduino.

Deberás conectarlos junto a las resistencias, para despues llevarlos a pines digitales de la pplaca Arduino.

Código del Circuito

#include int cm=0; long distancia,duracion; int posicion1=90; int posicion2=-90; Servo miServo; void setup() { Serial.begin (9600); pinMode(Pecho, INPUT); pinMode(Ptrig, OUTPUT); miServo.attach (13); } void loop() { digitalWrite (Ptrig,LOW); delayMicroseconds(2); digitalWrite(Ptrig,HIGH); delayMicroseconds(10); digitalWrite(Ptrig,LOW); duracion= pulseIn(Pecho, HIGH); distancia =(duracion/2)/29; if (distancia >=500 || distancia <=0){ Serial.println("---"); } else { Serial.print(distancia); Serial.println("cm"); digitalWrite(13,LOW); } if (distancia <=10 && distancia >=1){ miServo.write(posicion1); Serial.println("Obstaculo"); } else { miServo.write(posicion2); } delay(400); }
