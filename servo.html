Servo Motor

Uso del Servo Motor

📋 Descripción

En la práctica de hoy se realizará el movimiento de un Servo Motor.

🛠️ Materiales

  • Servo Motor
  • Arduino
  • Jumpers
1

Conexión

Señal → pin digital 9
VCC → 5V
GND → GND

2

Código del Circuito

#include <Servo.h> Servo servo; void setup() { servo.attach(9); } void loop() { servo.write(0); delay(1000); servo.write(90); delay(1000); servo.write(180); delay(1000); }
